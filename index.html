Learn PHP

The below code is used to print a statement in PHP

echo "Hello World", " Learning php";

Instead of echo you can use print

print "Hello World" . " Learning php";

The "." here means concatination.

Creating a variable in php

$vari = "Harshal";
echo "Hello there, my name is ${vari}";

The name of the variables cannot start with the numericals

The following way is to make a comment in php. 

// echo "This is a comment in php";

Following are the Data Types in PHP


  • Integer
  • Floating point or float
  • String
  • Boolean
  • Array
  • Object
  • Resource
  • Null


$age = 19;
echo "My age is, {$age}"; // This is a type of integer

Arrays

$cars = array('bmw', 'lambourghini', 'mercedes');
$carsFav = array('bullet', 'admiral', 'alpha');
var_dump($cars);
echo "<br>";
var_dump($carsFav);
echo "<br>";
print_r($cars);
echo "<br>";
print_r($carsFav);

$merger = array_merge($cars, $carsFav);
print_r($merger);
